What is Visual Code Studio:

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE

What is Console:

Console is a screen where you display your output, and it is used for testing purposes. Before printing anything on the HTML page it is advisable to print on the console screen.

How to open Console screen:

Step 1: To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J (on a Mac) or Cmd +Shift +J (on Windows). As an alternative, you can right-click on the webpage and click "Inspect" to open the developer console.

Step 2: Click the "Console" tab in that window.
